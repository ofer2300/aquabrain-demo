# aquabrain-demo
Next.js 14 TypeScript starter with RTL/i18n (Hebrew), Tailwind CSS, app router demo workspace

## Features

### Project Timeline Management Widget
A comprehensive construction project timeline management interface featuring:
- **Compact Header Widget**: Displays planned vs actual timelines with 3 key milestones
- **Full-Screen Detailed View**: Complete Gantt chart with all project activities
- **Visual Deviation Tracking**: Color-coded status indicators and variance calculations
- **Interactive Timeline**: Click to expand, hover for details

See [TIMELINE_WIDGET_README.md](./TIMELINE_WIDGET_README.md) for complete documentation.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000/demo` to see the timeline widget in action.
