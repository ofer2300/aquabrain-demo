# aquabrain-demo

Next.js 14 TypeScript starter with RTL/i18n (Hebrew), Tailwind CSS, app router demo workspace

---

## סקירה כללית

AquaBrain הוא פרויקט דמו מתקדם לפיתוח אפליקציות Next.js בעברית, המציע בסיס עבודה מוכן, רספונסיבי ומותאם RTL, מתוך מטרה לייעל פיתוח אפליקציות ווב לישראלים, ולאפשר תוצרים איכותיים בזמן קצר.

הפרויקט מבוסס על Next.js 14 עם תמיכה מלאה בעברית (RTL), TypeScript, עיצוב רספונסיבי ב-Tailwind CSS, ומספק תשתית מוכנה לעבודה עם App Router, מצב כהה/בהיר, ודגש על Type Safety.

## כיצד הדברים היו נראים ללא AquaBrain?

אם הפרויקט לא היה קיים:

- **פיתוח מאפס**: כל יישום היה דורש פיתוח תשתיתי מלא להתמודדות עם עברית ו-RTL
- **עלויות גבוהות**: תוספת משמעותית בזמן, עלויות ותלות במפתחי תשתית מנוסים
- **סיכון לטעויות**: הסיכון לטעויות והורדת איכות המוצר עולה משמעותית
- **פיתוח איטי**: תהליך פיתוח ארוך יותר וחוויה שאינה תואמת את הצרכן המקומי
- **אתגרי RTL**: צורך להתמודד ידנית עם כל בעיות הכיוון, עיצוב ונגישות

### השפעה על כל סוג פרויקט:

**אתרים עסקיים**: היית מתחיל מ-0 או מתבניות מחו"ל, מה שמקשה על התאמה לעברית, נגישות RTL ואינטגרציה עם כלים מקומיים.

**מערכות CMS**: נדרש להקים בעצמך תמיכה מלאה בעברית, בנייה של רכיבי ניהול והגדרות RTL מאפס.

**חנויות מקוונות**: התמודדות לבד עם עיצוב מתקדם, פתרונות סליקה ותמיכה RTL; ההתחלה ללא בסיס גוזלת זמן ויוצרת תקלות.

**דשבורדים**: אין בסיס מוכן לגרפים, אינטגרציה RTL ודוגמאות לקומפוננטים מותאמים.

## על רכיב ה-Router

Router הוא חלק מהאפליקציה (לא מהדפדפן) שמנהל ניווט פנימי בין דפים/רכיבים:

- מאפשר מעבר חלק ומהיר בין מסכים ללא טעינה מחדש של הדף
- תורם לחוויית SPA (Single Page Application) מודרנית
- מנהל כתובות URL פנימיות ומציג תוכן מתאים לכל Route
- שונה מהניווט הקלאסי של הדפדפן (Full Page Reload)

## טכנולוגיות עיקריות

- **Next.js 14.2.5** - Framework מתקדם לפיתוח React
- **React 18.3** - ספריית UI מודרנית
- **TypeScript 5.x** - Type Safety ופיתוח בטוח
- **Tailwind CSS 3.4** - עיצוב רספונסיבי מהיר
- **App Router** - ניתוב מתקדם ו-Server Components
- **RTL Support** - תמיכה מלאה בעברית וכיווניות ימין לשמאל

## התחלה מהירה

```bash
# התקנת תלויות
npm install

# הרצה במצב פיתוח
npm run dev

# בנייה לפרודקשן
npm run build

# הרצת פרודקשן
npm start
```

האתר יהיה זמין ב: `http://localhost:3000`
